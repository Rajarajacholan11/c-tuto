Chapter 7: Arrays

An array is a collection of similar elements stored in contiguous memory.

int marks[90];
char name[20];
float percentile[90];

Access using:

marks[0] = 33;
scanf("%d", &marks[0]);
printf("%d", marks[0]);

Pointer Arithmetic

Pointer increment depends on data type size:

int *a = &x; // 4 bytes
a++; // points to next int location

char *b = &y; // 1 byte
b++;

// Operations: +, -, ptr1 - ptr2, comparison

Arrays with Pointers

int arr[5] = {9, 8, 7, 6, 5};
int *ptr = arr; // or &arr[0]
printf("%d", *ptr);    // 9
ptr++;

Arrays can be passed to functions:

void printArray(int arr[], int size);

Multidimensional Arrays

int arr[3][2] = {
  {1, 4},
  {7, 9},
  {11, 22}
};

printf("%d", arr[0][1]); // 4

Stored in contiguous memory blocks.

Chapter 8: Strings

A string is a character array ending with a null character '\0'.

  char s[] = {'H', 'A', 'R', 'R', 'Y', '\0'};
char s1[] = "HARRY";

String Input & Output

char st[50];
scanf("%s", st); // one word only

gets(st);  // multi-word input
puts(st);  // output

Strings using pointer:

char *ptr = "Harry";

String Library Functions

  • strlen(s) – returns string length
  • strcpy(dest, src) – copy
  • strcat(s1, s2) – concatenate
  • strcmp(s1, s2) – compare

#include <string.h> is required.

Chapter 9: Structures

A structure is a user-defined type that can hold variables of different data types.

struct employee {
  int code;
  float salary;
  char name[10];
};

struct employee e1;
e1.code = 100;
e1.salary = 71.22;
strcpy(e1.name, "Harry");
