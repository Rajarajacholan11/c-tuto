Chapter 10: File I/O

To persist data between program executions, use files.

FILE *ptr;
ptr = fopen("data.txt", "r");

FILE is a structure, pointer is used to communicate with files.

Modes

  • "r" – read
  • "w" – write (overwrite)
  • "a" – append
  • "rb" / "wb" – binary

Reading & Writing Files

    fscanf(ptr, "%d", &num);    // Read int from file
fprintf(ptr, "%d", num);    // Write int to file

char ch;
while ((ch = fgetc(ptr)) != EOF) {
  putchar(ch);
}

fputc('A', ptr);

Always close files:

    fclose(ptr);

Chapter 11: Dynamic Memory Allocation

Memory allocated during runtime using:

  • malloc()
  • calloc()
  • free()
  • realloc()
    int *ptr = (int *) malloc(5 * sizeof(int)); 
if (ptr == NULL) { 
  printf("Memory allocation failed\n"); 
} else {
  printf("Memory allocated successfully\n");
}

calloc() and realloc()

    float *arr = (float *) calloc(5, sizeof(float));
arr = realloc(arr, 10 * sizeof(float));

calloc initializes to 0, malloc doesn't. free() deallocates.
