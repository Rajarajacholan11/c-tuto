What is Programming?

Computer programming is a medium for us to communicate with computers. Just like we use Hindi or English to communicate with each other, programming is a way to deliver our instructions to the computer.

What is C?

C is a programming language developed by Dennis Ritchie at AT&T Bell Labs, USA in 1972. It is one of the oldest and finest programming languages.

Uses of C:

  • Major parts of Windows, Linux, and other operating systems are written in C.
  • Used to write driver programs for devices like tablets, printers, etc.
  • Used to program embedded systems (e.g., microwaves, cameras).
  • Used in game development where performance is critical.

Variables

A variable is a container which stores a value.

a = 3;
b = 4.7;
c = 'A';

Rules for Naming Variables:

  • First character must be an alphabet or underscore (_)
  • No commas or blanks allowed
  • No special symbols allowed (except _)
  • Case sensitive

Constants

A constant is an entity whose value doesn’t change. A variable, in contrast, can be changed during execution.

Types of Constants:

  • Integer Constant: -1, 6, 7, 9
  • Real Constant: -322.1, 2.5, 7.0
  • Character Constant: 'a', '$', '@' (enclosed in single quotes)

Keywords in C

Keywords are reserved words whose meaning is already known to the compiler. There are 32 keywords in C:

Our First C Program

#include <stdio.h>

int main() {
    printf("Hello, I am learning C with Harry");
    return 0;
}

File: first.c

Basic Structure of a C Program

  • Every program starts from the main() function.
  • Statements end with a semicolon.
  • C is case-sensitive.
  • Instructions are executed in the written order.

Comments

Used to clarify parts of the program. Ignored during execution.

  • Single-line comment: // This is a comment
  • Multi-line comment: /* This is a multi-line comment */

Compilation and Execution

A compiler converts human-readable C code into machine code (.exe). It performs basic checks and builds an executable program.

Library Functions

C provides built-in functions to perform tasks. Example:

printf("This is %d", i);

%d → Integer, %f → Real numbers, %c → Characters
