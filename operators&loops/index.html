Usage of Logical Operators

  • && (AND) is true only when both conditions are true.
  • || (OR) is true when at least one condition is true.
  • ! (NOT) reverses the truth value.

Example: !(3 == 3) → false, !(3 != 3) → true

Else If & Ternary Operator

Else-if Ladder: Useful for checking multiple conditions.

Ternary Operator: Short-hand if-else:

        condition ? expression_if_true : expression_if_false;

Switch Case Statement

switch (expression) {
  case 1:
    // code
    break;
  case 2:
    // code
    break;
  default:
    // code
}
  • Can use int or char values
  • Cases can be in any order
  • Can nest switch statements (though rarely used)

Chapter 4: Loop Control Instructions

Loops are used to execute a block of code repeatedly.

  • while loop
  • do-while loop
  • for loop

While Loop

int i = 0;
while (i < 10) {
  printf("i = %d\n", i);
  i++;
}

Note: Infinite loop if the condition never becomes false.

Do-While Loop

int i = 0;
do {
  printf("%d\n", i);
  i++;
} while (i < 5);

Executes the block at least once.

For Loop

for (int i = 0; i < 3; i++) {
  printf("%d\n", i);
}

Can increment or decrement:

for (int i = 5; i > 0; i--) {
  printf("%d\n", i);
}

Break and Continue

Break:

Exits the loop immediately.

for (int i = 0; i < 100; i++) {
  if (i == 5) break;
  printf("%d\n", i);
}

Continue:

Skips current iteration.

int skip = 5;
for (int i = 0; i < 10; i++) {
  if (i == skip) continue;
  printf("%d\n", i);
}
