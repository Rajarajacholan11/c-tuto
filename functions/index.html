Chapter 5: Functions

Functions allow code reuse and logical separation.

void display();

  int main() {
    display(); // function call
    return 0;
  }

  void display() {
    printf("Hi I am display");
  }

Passing Values to Functions

int sum(int a, int b) {
    return a + b;
  }

  int main() {
    int result = sum(2, 3); // result = 5
  }

Note: Parameters are variables in definition; arguments are values passed during the call.

Recursion

A function that calls itself is recursive.

int factorial(int n) {
    if (n == 0 || n == 1) return 1;
    return n * factorial(n - 1);
  }

Base condition avoids infinite recursion.

Chapter 6: Pointers

A pointer stores the memory address of another variable.

  int a = 8;
int *p = &a;

printf("Address: %u\n", p);     // address of a
printf("Value: %d\n", *p);     // value at that address (8)

Pointer to Pointer

int a = 5;
  int *p = &a;
  int **q = &p;

  printf("%d\n", **q); // prints 5

Used in advanced memory manipulation and nested structures.

Call by Value vs Reference

Call by Value

Original value not affected.

void change(int x) {
    x = 77;
  }

Call by Reference

void swap(int *x, int *y) {
    int temp = *x;
    *x = *y;
    *y = temp;
  }

Calling with swap(&a, &b) swaps values in-place.
