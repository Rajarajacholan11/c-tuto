Types of Variables

  • int a = 3; – Integer variable
  • float a = 7.7; – Real variable
  • char a = 'B'; – Character variable

Receiving Input from the User

To receive input in C, we use the scanf function.

int i;
scanf("%d", &i);

The & is the “address of” operator. It tells the compiler to store the user input at the address of the variable.

Chapter 2: Instructions & Operators

A C program contains instructions. Types:

  • Type Declaration Instruction: int a;, float b;
  • Arithmetic Instruction
  • Control Instruction

You can also assign multiple variables at once:

int a, b, c, d;
a = b = c = d = 30;

Arithmetic Operators

  • +, -, *, /, % (modulo for remainder)

Note: % cannot be used with float.

For exponents, C does not have an operator. Use pow(x, y) from <math.h>.

Type Conversion

Rules:

  • int + int → int
  • int + float → float
  • float + float → float

Example: int k = 3.0 / 9; → 0, because float gets demoted.

Operator Precedence

C uses operator precedence and associativity rules instead of BODMAS.

  • *, /, % have higher precedence than +, -
  • Associativity for most operators is left to right

Control Instructions

  • Sequence Control Instruction
  • Decision Control Instruction
  • Loop Control Instruction
  • Case Control Instruction

Chapter 3: Conditional Instructions

Conditional logic allows executing instructions based on whether a condition is met.

Types:

  • If-Else Statement
  • Switch Statement

If-Else & Relational Operators

int a = 23;
if (a > 18) {
    printf("You can drive\n");
}

Relational operators:

  • == equal to
  • != not equal to
  • >=, <=, >, <

Logical operators:

  • && AND
  • || OR
  • ! NOT
